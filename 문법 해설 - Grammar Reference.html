문법 해설 — Grammar Reference

Apéndice del libro: explicaciones detalladas por unidad

1과 — 언어 교환

1. N(이)라고 하다

Se usa para citar o nombrar algo. "라고" se añade a citas o nombres.

발침 X: N + 라고 하다 → 스티븐이라고 합니다

발침 O: N + 이라고 하다 → 김민수라고 합니다

→ Ver Grammar 1-1 U1

2. V-(으)려고

Se añade al final de una cláusula para expresar intención o propósito.

발침 X: V + 려고 → 가려고

발침 O: V + 으려고 → 먹으려고

💡 -(으)려고 vs -(으)려고 하다

- -(으)려고 sola: liga la intención con la acción siguiente en la misma frase

→ 한국어를 배우려고 한국에 왔어요. (Vine a Corea para aprender coreano.)

- -(으)려고 하다 al final: la intención ES el predicado principal

→ 한국에 가려고 해요. (Tengo intención de ir a Corea.)

→ Ver Grammar 1-2 U1

3. V-거나

Se añade a verbos para indicar opciones ("o... o..."). La raíz nunca cambia.

→ 주말에 영화를 보거나 음악을 들어요. (Los fines de semana veo películas o escucho música.)

→ Ver Grammar 2-1 U1

4. N(이)나

Se añade a sustantivos para indicar opciones entre cosas ("o").

발침 X: N + 나 → 사과나

발침 O: N + 이나 → 책이나

→ Ver Grammar 2-2 U1

2과 — 취미가 뭐예요?

1. V-는 것

"-는 것" se añade a raíces verbales y convierte el verbo en frase nominal.

발침 X: 오다 → 오는 것

발침 O: 먹다 → 먹는 것

Puede funcionar como sujeto (con 이/가) o como objeto (con 을/를):

→ 운동하는 것이 건강에 좋아요. (Hacer ejercicio es bueno para la salud.)

→ 저는 운동하는 것을 좋아해요. (Me gusta hacer ejercicio.)

💡 En lengua hablada, "것" se abrevia con la partícula:

것이 → 게 / 것을 → 걸 / 것은 → 건

→ 운동하는 게 좋아요 (más natural que 것이)

💡 -기 reemplaza 것 en lengua hablada también:

→ 일찍 일어나는 것(→ 기) 힘들어요.

→ 저는 영화 보는 것을(→ 걸) 좋아해요.

→ 외국어를 배우는 것은(→ 건) 재미있어요.

→ Ver Grammar 1-1 U2

2. V-(으)ㄹ 줄 알다/모르다

Se usa para expresar si sabes o no sabes hacer algo (habilidad).

발침 X: V + ㄹ 줄 알다 → 수영하ㄹ 줄 알아요

발침 O: V + 을 줄 알다 → 읽을 줄 알아요

💡 Diferencia con V-(으)ㄹ 수 있다/없다 (posibilidad vs habilidad):

- 줄 알다 = saber cómo (habilidad aprendida)

- 수 있다 = poder (posibilidad según circunstancias)

→ Ver Grammar 1-2 U2

3. V-(으)ㄴ N

"(으)ㄴ" se añade a raíces verbales para modificar un sustantivo (pasado).

발침 X: 가다 → 간 N

발침 O: 먹다 → 먹은 N

ㄹ irregular: 만들다 → 만든 N

→ Ver Grammar 2-1 U2

4. A/V-지 않다

"-지 않다" se añade a raíces de adjetivos y verbos para negarlos.

형용사: 크다 → 크지 않다 / 작다 → 작지 않다

동사: 가다 → 가지 않다 / 먹다 → 먹지 않다

💡 못 부정형 (otra forma de negación):

"못 하다" = la forma de negación con 못 → 나는 한자를 읽지 못합니다

= 어제는 바빠서 친구들을 만나지 못했어요

→ Ver Grammar 2-2 U2

3과 — 콘서트에 가 봤어요?

1. V-아/어 보다

"-아/어 보다" se añade a raíces verbales para expresar experiencia pasada o intento.

아/오 → 아 보다: 가다 → 가 보다

otras → 어 보다: 먹다 → 먹어 보다

하다 → 해 보다: 요리하다 → 요리해 보다

→ 제주도에 가봤어요. (He ido a Jeju.)

→ 번지 점프를 해보세요. (Prueba el bungee jumping.)

→ Ver Grammar 1-1 U3

2. N 동안

"동안" se añade a sustantivos de tiempo para expresar duración.

발침 X/O: N + 동안 (la forma no cambia nunca)

→ 일주일 동안 / 한 달 동안 / 3년 동안

→ 얼마 동안 한국어를 공부했어요? (¿Cuánto tiempo has estudiado coreano?)

→ 방학 동안 제주도에 갔다 왔어요. (Durante las vacaciones fui a Jeju.)

💡 Días nativos 1–10 (solo existen hasta 10, del libro p.239):

하루(1) 이틀(2) 사흘(3) 나흘(4) 닷새(5) 엿새(6) 이레(7) 여드레(8) 아흐레(9) 열흘(10)

→ Ver Grammar 1-2 U3

3. A-(으)ㄴ데, V-는데 / N인데

Se usa para dar información de fondo/contexto antes de decir algo — contraste, trasfondo, o preámbulo a una pregunta.

형용사: A + (으)ㄴ데 → 바쁜데, 좋은데

동사: V + 는데 → 가는데, 먹는데

명사: N + 인데 → 학생인데

→ 저 학생인데요, 질문이 있어요. (Soy estudiante, tengo una pregunta.)

→ 내일 바쁜데, 같이 갈 수 있어요? (Mañana estoy ocupado, ¿puedes ir conmigo?)

→ 이 음식을 먹어 봤는데, 정말 맛있어요. (He comido esta comida y está muy rica.)

→ 이제 집에 가는데, 오늘 재미있었어요. (Me voy a casa ya, hoy fue divertido.)

💡 Muy versátil: puede ser contraste ("pero..."), contexto ("resulta que..."), o trasfondo para una pregunta. El tono lo decide el contexto.

→ Ver Grammar 2-1 U3 · Grammar 2-2 U3

4. V-(으)ㄹ N (modificador nominal futuro)

"(으)ㄹ" se añade a raíces verbales para modificar un sustantivo con sentido futuro o prospectivo.

발침 X: V + ㄹ + N → 갈 곳 (el lugar al que iré)

발침 O: V + 을 + N → 먹을 음식 (la comida que voy a comer)

→ 내일 갈 곳이 어디예요? (¿Adónde vas a ir mañana?)

→ 오늘 먹을 음식을 만들었어요. (Preparé la comida que vamos a comer hoy.)

→ 친구한테 줄 선물을 샀어요. (Compré un regalo que le daré a un amigo.)

💡 Resumen de los 3 modificadores nominales:

V-(으)ㄴ N → pasado: 어제 먹은 음식 (la comida que comí)

V-는 N → presente: 매일 먹는 음식 (la comida que como)

V-(으)ㄹ N → futuro: 내일 먹을 음식 (la comida que comeré)

→ Ver Grammar 2-3 U3

5. V-(으)ㄹ 것 같다 / V-는 것 같다

Se usa para expresar suposición o apariencia ("parece que...").

형용사: A + (으)ㄹ 것 같다 → 추울 것 같다 (parece que hará frío)

동사 presente: V + 는 것 같다 → 가는 것 같다 (parece que va)

동사 pasado: V + (으)ㄴ 것 같다 → 간 것 같다 (parece que fue)

→ 날씨가 추울 것 같아요. (Parece que hará frío.)

→ 그 사람이 한국 사람인 것 같아요. (Parece que esa persona es coreana.)
