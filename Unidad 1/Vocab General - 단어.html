Unidad 1 — Vocab General

Información personal (formularios / presentaciones)

성명 — nombre completo (apellido + nombre)

성별 — sexo / género

생년월일 — fecha de nacimiento (생년 = año de nacimiento / 월 = mes / 일 = día)

직업 — trabajo / ocupación

국적 — nacionalidad

종교 — religión

연락처 — información de contacto

주소 — dirección

Religiones y lugares de culto

불교 — budismo

기독교 / 천주교 — cristianismo protestante / catolicismo

이슬람교 — islam

절 / 사원 — templo budista

교회 — iglesia (protestante)

성당 — catedral / iglesia católica

"Quizás / tal vez"

아마 — probablemente / quizás (~70%) (아마 내일 올 거예요 = probablemente vendrá mañana)

아마도 — tal vez (un poco menos seguro) (아마도 맞을 거예요 = tal vez sea correcto)

혹시 — por si acaso / ¿quizás? (en preguntas) (혹시 알아요? = ¿acaso lo sabes?)

Línea de tiempo

Días

그저께 — anteayer

어제 — ayer

오늘 — hoy

내일 — mañana

모레 — pasado mañana

Semanas / meses / años

지난주 (semana pasada) → 이번 주 (esta semana) — 다음 주 (la próxima semana)

지난달 (mes pasado) → 이번 달 (este mes) — 다음 달 (el próximo mes)

작년 (año pasado) → 올해 (este año) — 내년 (el próximo año)

Pasado / Presente / Futuro (general)

과거 — pasado

현재 — presente

미래 — futuro
