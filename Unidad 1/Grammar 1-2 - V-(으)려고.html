Grammar 2 — V-(으)려고

Qué hace: Expresa el propósito o intención de una acción.

Español: "para...", "con la intención de..."

⚠️ El sujeto de las dos acciones debe ser el mismo.

Forma

Vocal → 려고 — 이야기하려고

Consonante → 으려고 — 먹으려고

Diálogo del libro

A: 왜 한국어를 공부하세요?
(¿Por qué estudias coreano?)

>

B: 한국 친구들과 이야기하려고 공부해요.
(Estudio para hablar con amigos coreanos.)

Ejemplos del libro

편지를 보내려고 우체국에 갔어요.
- 편지를 = carta (objeto) / 보내려고 = para enviar / 우체국에 = a la oficina de correos / 갔어요 = fui
- "Fui a la oficina de correos para enviar una carta."
저녁에 먹으려고 과일을 샀어요.
- 저녁에 = por la noche / 먹으려고 = para comer / 과일을 = fruta / 샀어요 = compré
- "Compré fruta para comer por la noche."

Del texto de lectura (Unidad 1)

중남미를 여행하려고 스페인어를 공부하고 있습니다.
"Estoy estudiando español para viajar por Latinoamérica."

Vocab útil para los ejercicios

서점 — librería

책을 사다 — comprar un libro

돈을 모으다 — ahorrar dinero

전화기 — teléfono

부엌 — cocina

물을 마시다 — beber agua

일찍 일어나다 — levantarse temprano

운동하다 — hacer ejercicio

팝콘 — palomitas

Ejercicios

1. Conecta las dos acciones con -(으)려고:

1. 물을 마시다 + 부엌에 가요 → (voy a la cocina para beber agua)

2. 친구를 만나다 + 카페에 갔어요 → (fui al café para encontrarme con un amigo)

3. 한국어를 배우다 + 책을 샀어요 → (compré un libro para aprender coreano)

4. 운동하다 + 일찍 일어났어요 → (me levanté temprano para hacer ejercicio)

5. 영화를 보다 + 팝콘을 샀어요 → (compré palomitas para ver la película)

2. Responde libremente:

왜 한국어를 공부해요? (¿Por qué estudias coreano?)

→ 저는 ___(으)려고 한국어를 공부해요.

3. Traduce al coreano:

1. Fui a la librería para comprar un libro.

2. Estudio coreano para viajar a Corea.

3. Ahorré dinero para comprar un teléfono.
