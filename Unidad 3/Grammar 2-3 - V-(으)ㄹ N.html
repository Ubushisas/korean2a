Grammar 2-3 — V-(으)ㄹ N

(문법과 표현 2 — Track 27)

Unidad 3 — 콘서트에 가 봤어요?

📌 Lo que hace: Describe un sustantivo con algo que va a pasar — "el regalo que voy a dar"

🇰🇷 Estructura: verbo (futuro) + ㄹ/을 + sustantivo

🇪🇸 Español: "El regalo que voy a dar" → 친구한테 줄 선물

🇬🇧 English: "The gift I'm going to give" → 친구한테 줄 선물

Forma

발침 X: V + ㄹ + N → 갈 곳, 볼 영화

발침 O: V + 을 + N → 먹을 음식, 읽을 책

ㄹ irregular: raíz ya termina en ㄹ → 만들 것, 살 집

Diálogo del libro (Track 27)

A: 어디 가세요?

(¿Adónde va?)


B: 친구한테 줄 선물을 사러 가요.

(Voy a comprar un regalo que le voy a dar a un amigo.)

친구한테 줄 거예요 (acción futura) → 선물 (el sustantivo que describe)

💡 사러 가요 — nuevo patrón: V-러 가다/오다 = ir/venir para hacer algo. Solo con verbos de movimiento. 사러 가요 = "voy a comprar" (lit. voy para comprar).

Ejemplos del libro

내일은 만날 시간이 없어요.

Mañana no tengo tiempo para quedar.

내일 만날 거예요 (acción futura) → 시간 (el sustantivo que describe)

교실에 앉을 자리가 없어요.

No hay sitio para sentarse en el aula.

앉을 거예요 (acción futura) → 자리 (el sustantivo que describe)

저녁에 먹을 빵 좀 사 오세요.

Por favor trae un poco de pan para comer esta noche.

저녁에 먹을 거예요 (acción futura) → 빵 (el sustantivo que describe)

Los 3 modificadores — resumen completo

V-(으)ㄴ N → pasado: 어제 먹은 음식 (la comida que comí ayer)

V-는 N → presente: 매일 먹는 음식 (la comida que como todos los días)

V-(으)ㄹ N → futuro: 내일 먹을 음식 (la comida que comeré mañana)

💡 Truco: ㄴ=pasado (ya pasó), 는=presente (está pasando), ㄹ=futuro (va a pasar)

Vocab útil del libro

선물을 사다 — comprar un regalo

친구한테 주다 — darle a un amigo

제출하다 — entregar (tarea, documento)

날짜 — fecha

배우다 — aprender

살다 — vivir → 살 집 (la casa en que viviré)

여행하다 — viajar → 여행할 곳 (el lugar al que viajaré)

Ejercicios

1. Completa con -(으)ㄹ N (rellena el espacio):

저는 내일 친구들과 놀러 갑니다. 친구들과 놀러 갈 곳은 여의도공원입니다. 우리는 내일 여의도공원에서 점심을 먹을 겁니다. 저는 내일 친구들하고 1) \_\_\_ 음식과 2) \_\_\_ 것을 준비해야 합니다. 내일 낮에는 공원에서 놀고 저녁에는 쇼핑을 하려고 합니다. 다음 주에 여행을 가는데 비행기에서 3) \_\_\_ 책도 사고 친구에게 4) \_\_\_ 선물도 사야 합니다.

Pista: cada espacio necesita un verbo conjugado con -(으)ㄹ

  1. \_\_\_ 음식 → ¿qué van a hacer con la comida?
  2. \_\_\_ 것 → ¿qué van a hacer con las cosas que preparan?
  3. \_\_\_ 책 → ¿qué vas a hacer con el libro en el avión?
  4. \_\_\_ 선물 → ¿qué vas a hacer con el regalo?
