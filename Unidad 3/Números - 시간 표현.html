Números y Tiempo — 시간 표현

Cuándo usar números sino-coreanos (한자어) vs nativos coreanos (고유어)

La regla maestra

Números nativos (고유어): 하나, 둘, 셋, 넷, 다섯, 여섯, 일곱, 여덟, 아홉, 열...

→ Se usan con horas y con contadores de objetos/personas (개, 명, 잔...)

Números sino-coreanos (한자어): 일, 이, 삼, 사, 오, 육, 칠, 팔, 구, 십...

→ Se usan con minutos, segundos, meses, años, dinero, pisos, fechas

Tiempo — tabla completa

Horas (시)

→ NATIVO — 한 시, 두 시, 세 시, 네 시, 다섯 시... 열두 시

(la hora del reloj siempre nativo)

Minutos (분)

→ SINO — 일 분, 이 분, 십오 분, 삼십 분...

(los minutos siempre sino-coreano)

Segundos (초)

→ SINO — 일 초, 오 초, 삼십 초...

Días (일)

→ SINO para fechas: 일월 일 일 (1 de enero), 이십오 일 (día 25)

→ NATIVO especial para duración de días 1–10 (solo existen hasta 10):

1일 = 하루 / 2일 = 이틀 / 3일 = 사흘 / 4일 = 나흘 / 5일 = 닷새

6일 = 엿새 / 7일 = 이레 / 8일 = 여드레 / 9일 = 아흐레 / 10일 = 열흘

→ Del 11 en adelante → SINO: 십일 일, 십이 일, 이십 일...

→ Sino-coreano también válido para 1–10: 일 일, 이 일, 삼 일 동안... (menos literario)

Semanas (주/주일)

→ SINO — 일주일 (1 semana), 이주일 (2 semanas), 삼 주 동안...

Meses (달 / 개월)

→ NATIVO para el mes como unidad de tiempo: 한 달, 두 달, 세 달...

→ SINO con contador 개월: 일 개월, 이 개월, 육 개월...

→ Los dos son correctos, 달 es más coloquial

Año (년)

→ SINO — 일 년, 이 년, 삼 년 동안...

Con 동안 — ejemplos directos

한 시간 동안 — durante una hora (시간 = hora como duración → NATIVO: 한)

삼십 분 동안 — durante 30 minutos (분 → SINO)

일주일 동안 — durante una semana (주일 → SINO)

한 달 동안 — durante un mes (달 → NATIVO: 한)

두 달 동안 — durante dos meses (달 → NATIVO: 두)

일 개월 동안 — durante un mes (개월 → SINO: 일)

삼 년 동안 — durante tres años (년 → SINO)

5일 동안 — durante 5 días (일 → SINO)

Truco para no confundirse

Unidad — Sistema Ejemplo

시 (hora del reloj) — NATIVO 두 시 = las 2:00

시간 (horas de duración) — NATIVO 두 시간 = 2 horas

분 (minutos) — SINO 이십 분 = 20 min

초 (segundos) — SINO 삼십 초 = 30 seg

일 (días) — SINO 삼 일 = 3 días

달 (meses, coloquial) — NATIVO 두 달 = 2 meses

개월 (meses, formal) — SINO 이 개월 = 2 meses

주/주일 (semanas) — SINO 이주일 = 2 semanas

년 (años) — SINO 삼 년 = 3 años

💡 Regla práctica: Si la palabra termina en 시 o 달 → usa números nativos. Todo lo demás (분, 초, 일, 주, 개월, 년) → usa sino-coreano.

Números nativos hasta 99

1–10: 하나, 둘, 셋, 넷, 다섯, 여섯, 일곱, 여덟, 아홉, 열

11–19: 열하나, 열둘... 열아홉

20–90: 스물(20), 서른(30), 마흔(40), 쉰(50), 예순(60), 일흔(70), 여든(80), 아흔(90)

💡 Con contador, 하나→한, 둘→두, 셋→세, 넷→네: 한 시 (no ~~하나 시~~), 두 달 (no ~~둘 달~~)

Números sino-coreanos hasta 99

1–10: 일, 이, 삼, 사, 오, 육, 칠, 팔, 구, 십

11–99: 십일(11), 이십(20), 삼십(30), 사십(40)... 구십구(99)
