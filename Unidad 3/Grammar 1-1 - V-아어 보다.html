Grammar 1-1 — V-아/어 보다

(문법과 표현 1 — Track 22)

Qué hace: Expresa experiencia pasada o que intentas/pruebas hacer algo.

Español: "haber + participio" / "probar a hacer"

English: "have you ever...?" / "try doing..."

Forma

아 / 오 → 아 보다 — 가다 → 가 봐요

Otras → 어 보다 — 먹다 → 먹어 봐요

하다 → 해 보다 — 요리하다 → 요리해 봐요

Tiempos:

  • 경험 (experiencia pasada) → 아/어 봤어요
  • 시도 (intento presente) → 아/어 보세요 (inténtalo / pruébalo)

Diálogo del libro

A: 제주도에 가 봤어요?
(¿Has ido a Jeju?)

>

B: 네, 가 봤어요.
(Sí, he ido.)

Ejemplos del libro

작년 여름에 부산에 가 봤어요.
- 작년 여름에 = el verano pasado
- 가 봤어요 = fui (y fue una experiencia)
- "El verano pasado fui a Busan."
히엔 씨, 바다에서 낚시해 봤어요?
- 바다에서 = en el mar
- 낚시하다 = pescar → 낚시해 봤어요 = ¿has pescado?
- "Hien, ¿has pescado en el mar?"
한국에 와서 김치를 처음 먹어 봤어요.
- 한국에 와서 = habiendo venido a Corea
- 처음 = por primera vez
- 먹어 봤어요 = probé (por primera vez)
- "Vine a Corea y probé el kimchi por primera vez."

Diferencia clave: 봤어요 vs 보세요

아/어 봤어요**** → experiencia pasada — 김치를 먹어 봤어요 = he probado el kimchi

아/어 보세요**** → sugerencia / intento — 김치를 먹어 보세요 = prueba el kimchi

Vocab útil

제주도 — isla Jeju

부산 — Busan (ciudad)

작년 여름 — el verano pasado

바다에서 — en el mar

낚시하다 — pescar

처음 — por primera vez

얼마나 — cuánto / cuánto tiempo

구경하다 — recorrer / pasear mirando

Ejercicios

1. Conjuga con -아/어 봤어요:

1. 한국 음식을 먹다 → ¿Has comido comida coreana?

2. 한복을 입다 → ¿Has vestido hanbok?

3. 태권도를 배우다 → ¿Has aprendido taekwondo?

4. 한국 드라마를 보다 → ¿Has visto un drama coreano?

2. Responde libremente:

제주도에 가 봤어요? (¿Has ido a Jeju?)

→

한국 음식을 먹어 봤어요? (¿Has comido comida coreana?)

→

3. Transforma en sugerencia (-아/어 보세요):

1. 이 영화를 보다 → prueba ver esta película

2. 김치찌개를 먹다 → prueba el kimchi jjigae
